
# Automatic Login Scraper

## Overview
This Python Selenium project allows you to:

- Automatically log into a web dashboard
- Scrape records, cards, or tables from the dashboard
- Automatically log out after scraping
- Use environment variables for safe credential management

## Features
- Auto-login using email and password
- Scrapes dashboard elements dynamically
- Handles modern web applications built with React, Vue, or similar frameworks
- Automatic logout to maintain session security
- Configurable for different dashboards by updating selectors
